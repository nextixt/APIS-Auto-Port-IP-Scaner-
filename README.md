# APIS-Auto-Port-IP-Scaner
This is the simple terminal-based scaner. You can use it for cybersecurity or just for fun!

# HOW TO USE

It has 2 variants of work:

1. SINGLE. It means you put and scan only one port.
   Example:
   in: SINGLE 127.0.0.1 443
   out: Port 443 is open
2. RANGE. It means you put and scan range of ports.
   Example:
   in: RANGE 127.0.0.1 443 445
   out: Port 443 is open
        Port 444 is open
        Port 445 is closed
